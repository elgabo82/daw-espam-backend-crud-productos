# Aplicación de Gestión de Catálogo de Productos
***
### Módulos

* express
* mysql2 
* sequelize
* cors
* dotenv
* jsonwebtoken
* bcryptjs

## Uso del respositorio

```bash
npm i express mysql2 sequelize cors dotenv jsonwebtoken bcryptjs

```


## Inicio del proyecto

### Opción 1
```bash
npm run dev

```

### Opción 2
```bash
nodemon.js src/server.js ó node

```
